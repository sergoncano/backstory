# How to write scripts for this program

- End every line with "\n" if you want the newline to be displayed.
- Then add another "\" at the end if you're actually newlining it in your editor, otherwise the macro will crash.
- If you want to separate sections by prompting the user for a key press, add a "\b".
- You _NEED_ to add a "\b" at the end or the last piece of text will not render.
